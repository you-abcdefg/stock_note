# StockNote

## サイト概要

### サイトテーマ
技術記事や学習メモをブックマーク・整理・共有できるエンジニア向けコミュニティサイトです。

### テーマを選んだ理由
未経験からソフトウェアエンジニアへの転職を目指し学習を進める中で、技術記事やQ&Aサイト、公式ドキュメントなど、参照する情報が分散し整理しきれないことに課題を感じていました。 
既存のブックマークサービスやQ&Aサイトは存在しますが、「自分用の備忘録として整理しつつ、後から検索しやすい形で蓄積できる」点に十分対応していないと感じました。 
そこで、記事のブックマークに対してユーザー自身が自由にタグ付けを行い、学習メモや備忘録として投稿・検索できるアプリケーションを制作することで、学習効率の向上と知識の定着を支援したいと考え、このテーマを選びました。

### ターゲットユーザ
- プログラミング学習中で、技術記事や参考情報を効率よく整理したい人
- 学習内容を備忘録として残し、後から振り返りたい人
- Q&Aサイトと個人メモを併用しており、情報管理に課題を感じている人

### 主な利用シーン
- 学習中に参考になった技術記事をブックマークし、タグ付けして整理したい時
- 過去に調べた内容やエラー解決方法を、キーワードやタグで素早く探したい時
- 学習した内容を自分用の備忘録として記事投稿したい時

## 機能概要（予定）
- ユーザー登録・ログイン機能
- 技術記事URLのブックマーク登録機能
- ブックマークへのオリジナルタグ付け機能
- タグ・キーワードによる検索機能
- 備忘録・学習メモの記事投稿機能
- 投稿・ブックマークのお気に入り管理機能
